Changing Access Control for Calendars and Address Books

sabre/dav features an extensive and highly customizable access control system. When writing nodes, collections and other objects from scratch, they allow for a very high degree of flexibility.

However, on the Cal- and CardDAV side, this functionality is not exposed in a way that's easily changed.

This is something we'd like to make easier in a future release, but until then, this is how access control is changed.

Relevant nodes and classes

What follows is a list of classes in SabreDAV, and which urls they (typically) handle:

url class
calendars Sabre\CalDAV\CalendarRootNode
calendars/[username] Sabre\CalDAV\UserCalendars
calendars/[username]/[calendar] Sabre\CalDAV\Calendar
calendars/[username]/[calendar]/[object.ics] Sabre\CalDAV\CalendarObject
addressbooks Sabre\CardDAV\AddressBookRoot
addressbooks/[username] Sabre\CardDAV\UserAddressBooks
addressbooks/[username]/[addressbook] Sabre\CardDAV\AddressBook
addressbooks/[username]/[addressbook]/[vcard.vcf] Sabre\CardDAV\Card

Overriding

If you are interested to change access to a calendar or addressbook, and make it read-only, you will need to extend either Sabre\CardDAV\AddressBook or Sabre\CalDAV\Calendar.

So pick a PHP namespace for your application, in our case we're picking MyGroupWareServer and override those classes.

The main function we need to change is the getACL function. For the address book, the default implementation looks like this:

public function getACL() {

    return array(
        array(
            'privilege' => '{DAV:}read',
            'principal' => $this->addressBookInfo['principaluri'],
            'protected' => true,
        ),
        array(
            'privilege' => '{DAV:}write',
            'principal' => $this->addressBookInfo['principaluri'],
            'protected' => true,
        ),

    );

}

So to modify this, we need to create a new class that extends the default address book class and returns something else:

namespace `MyGroupWareServer`

class AddressBook extends \Sabre\CardDAV\AddressBook {

    public function getACL() {

        return array(
            array(
                'privilege' => '{DAV:}read',
                'principal' => $this->addressBookInfo['principaluri'],
                'protected' => true,
            ),
        );

    }

}

We need to do the exact same for the Card class, so it will look a bit like this:

namespace `MyGroupWareServer`

class Card extends \Sabre\CardDAV\Card {

    public function getACL() {

        return array(
            array(
                'privilege' => '{DAV:}read',
                'principal' => $this->addressBookInfo['principaluri'],
                'protected' => true,
            ),
        );

    }

}
